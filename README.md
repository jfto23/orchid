# Orchid

_orchid_ is a space shooter game written in Rust.

![](https://github.com/jfto23/orchid/blob/master/orchid_pic.png)

## Build

If you want to play the game, you can clone the repository and do:

```
cargo run
```

## Controls

- `WASD`: Move

- `Space`: Shoot

- `J`: Special

- `K`: Shield

- `Q`: Quit

- `R`: Restart
